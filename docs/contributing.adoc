== Contributing & Development

To build Selenwright:

. Install https://golang.org/doc/install[Golang] 1.12 and above.

. Setup `$GOPATH` https://github.com/golang/go/wiki/GOPATH[properly]

. Clone Selenwright source:

    $ git clone https://github.com/aqa-alex/selenwright.git

. Go to project directory:

    $ cd selenwright

. Build source:

    $ go build
+
This will also fetch build dependencies.

. Run Selenwright:

    $ ./selenwright --help

[TIP]
====
To build Docker container type:

[source,bash]
----
$ mkdir -p dist
$ GOOS=linux GOARCH=amd64 CGO_ENABLED=0 go build -o dist/selenwright_linux_amd64
$ docker buildx build --pull --platform linux/amd64 -t selenwright:latest .
----

====

=== Documentation

Locally can be generated with:

[source,bash]
----
$ docker run --rm -v ./docs/:/documents/      \
    asciidoctor/docker-asciidoctor                \
    asciidoctor -D /documents/output/ index.adoc
----
