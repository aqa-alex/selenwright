== Setting Timezone

When used in Docker container Selenwright will have timezone set to *UTC*.
To set custom timezone pass `TZ` environment variable to Docker:

[source,bash,subs="attributes+"]
----
$ docker run -d --name selenwright                     \
    -p 4444:4444                                    \
    -e TZ=Europe/Berlin                             \
    -v /var/run/docker.sock:/var/run/docker.sock    \
    selenwright/hub:latest-release
----

NOTE: This sets the timezone for the Selenwright process itself (affects log timestamps, `/ping` response, etc.). To set timezone per browser session, use the `timeZone` capability — see <<Per-session Time Zone: timeZone>>.
