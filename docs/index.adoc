:title: Selenwright - Native Playwright and Selenium browser grid for Docker
:description: Run native Playwright WebSocket sessions and Selenium WebDriver tests in Docker containers. Selenwright launches isolated browser containers, supports VNC, video, logs, artifacts, authentication, and works on your own hardware or cloud VMs.
:favicon: img/favicon.png

Selenwright is a Docker-native browser automation grid with first-class Playwright WebSocket support and Selenium WebDriver compatibility.
It launches isolated browser containers for each session and keeps Playwright and Selenium workloads in one operational model. Please refer to https://github.com/aqa-alex/selenwright[GitHub repository] if you need source code.
